# Learning the Basics of TensorFlow/Keras and Tell-the-Time Network

In this project, along with Bashier Gulzar and Eleonora Roncaglia we learned the basics of TensorFlow via the Keras API in Python by building neural networks for classification tasks. 

In the first task we explore a multi-class perceptron for classifying the CIFAR-10 dataset and the Fashion MNIST dataset. 

In the second task, we build a convolutional neural network that can "tell the time" using three different approaches:
1. First treating the time as a classification task with different amounts of categories
2. Then, treating the time as a numeric value which can be predicted via regression
3. Finally, a two-head approach, where one part of the network focuses on predicting the minutes as a regression task and the other the hours as a classification task.
